## 13. RED FLAGS & WHAT TO AVOID

```
❌ DON'T:
- Buy backlinks
- Use link farms
- Comment spam on blogs
- Automated link software
- Private Blog Networks (PBNs)
- Reciprocal linking schemes
- Keyword-only anchor text (spam signal)

⚠️ RISKY:
- Exact match anchor text (use sparingly)
- Too many links in first week
- Links from irrelevant sites
- Low-quality directory submissions

✅ DO:
- Natural link growth
- Contextual backlinks
- Diverse anchor text
- Relevant high-authority sites
- White-hat link building only
```
